深海的大菠萝与肯德基

time limit: 1000ms

深海的大菠萝在寒假期间参加了数学建模。

在辛苦的数学建模后，深海的大菠萝会选择开车去吃肯德基，已知青岛有 家肯德基，包括学校在内我们可以把他们看作 个点，学校的编号为 ，其余肯德基编号为 到 ，深海的大菠萝最开始的位置在学校。

这 个点之间有 条双向道路相连接，由于青岛市部分街道在修路，并不是所有的路在一开始就能通行，我们用 表示点 和点 存在一条在 时间后（包括 时间）开放的双向道路。

现在深海大菠萝将询问你 次，每次询问你，如果他 时刻出发，能否到达所有的肯德基。

 

Input

第一行为一个正整数 ，表示测试样例的数目。

对于每个测试样例：

第一行为空格隔开的三个数 ，含义见题目描述。

接下来 行

每行为空格隔开的三个整数 ，含义如题所示

接下来 行

每行为一个正整数 ，表示询问深海的大菠萝 时刻出发，能否到达所有的肯德基

 

Output

对于每个测试样例：

共 行，即对 次询问的回答

如果可以到达，输出 “YES”（没有引号）

否则，输出“NO”（没有引号）

 

Sample Input

1
3 3 3
1 2 1
2 3 2
1 3 3
1
2
3

 

Sample Output

xxxxxxxxxx
NO
YES
YES
