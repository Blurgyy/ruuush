## Problem D    陶陶分质数

相信大家都知道 $NOIP$ 历史上有一道著名的题陶陶摘苹果，本题难度与那题相当，你们懂的

陶陶手上有一个数 $a$ ，他每次会从这个数里取走一个质因数 $x$ ，然后 $a$ 就变成了 $a/x$

他会一直重复上述操作，直到手中的数小于 $k$

他想知道他取走的所有质因数的和最大可以达到多少

__Input:__

一行中有两个整数，分别表示 $a$ 和 $k$

$2 \leq k \leq a \leq 40000$

__Output:__

输出可能的最大的和

__Sample Input:__

```
8 4
```

__Sample Output:__

```
4
```
